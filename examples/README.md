# Examples

Este diretório contém exemplos neutros para demonstrar a estrutura esperada pelo `kb` sem misturar corpus pessoal no repositório principal.

## Estrutura sugerida

```text
examples/
  raw/
    getting-started.md
```

## Uso sugerido

1. Crie um vault/corpus local fora deste repositório.
2. Aponte `KB_DATA_DIR` para esse diretório.
3. Copie os exemplos que quiser adaptar para o seu `raw/` local.

## Exemplo mínimo

Veja `examples/raw/getting-started.md`.
